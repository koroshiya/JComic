Comment utiliser JComic

	
	La plus grande partie des fonctionnalités de JComic ne requiert
	pas d’explications, mais voici les réglages que vous ne devriez
	pas changer sans compréhension de leurs fonctions. De même, si
	il y à quelque chose à propos de JComic que vous ne comprenez
	pas, référez-vous au manuel.
	
	Cette page fait office de guide rapide.
	Le manuel complet peut être trouvé à jcomic.japanzai.com

Comment débuter

	
	Il y a six boutons sur l’écran principal. Celui sur la gauche
	(l’icône d’un livre) vous guidera vers l’explorateur de fichiers.
	Vous pourrez alors naviguez à travers le contenu de votre appareil.
	Quand vous trouvez une image que vous souhaitez visualiser,
	cliquez simplement dessus.
	De même si vous trouvez l’archive dont vous souhaitez visualiser
	le contenu, un simple clique dessus suffit.

Fonctions par défaut

	
	En lisant avec JComic, par défaut :

	- Les images sont redimensionnées par leurs largeurs. Cela veut dire
	que les images larges seront réduites et les images hautes agrandies.

	- Cliquer deux fois sur une image chargée changera le grandissement.
	Le cycle de grandissement tourne entre Auto, 100%, ou n'importe
	lequel que vous auriez choisi par défaut dans les Réglages Généraux
	(Auto est par défaut).

	- Balayer vers la gauche affichera l'image supportée suivante.
	Balayer vers la droite affichera l'image supportée précédente.	

	- Balayer de haut en bas ne fait rien sauf si l'image que
	vous visualisez est trop grande pour être affichée. Dans ce cas,
	le balayage vertical est le même que le défilement vertical de l'image.

	- Changer l'orientation de votre appareil pendant la lecture (ex:
	le mettre sur la tranche) entrainera le redimensionnement de l'image
	vers la nouvelle largeur.
	Par exemple, si vous êtes en train de lire une BD en mode portrait
	(appareil vertical) et le tournez vers le mode paysage (appareil sur
	la tranche), l'image sera agrandie de manière à ce que sa largeur
	remplisse l'affichage en mode paysage, vous permettant de défiler
	verticalement pour afficher le reste de l'image.
	
	
Formats supportés

	
	JComic peut aussi utiliser d’autres formats mais les suivants sont
	complétement supportés.
	
	Formats d’images supportées : jpg, jpeg, png, gif, webp (Android 4.0+)

	Formats d’archive supportées : zip, rar, cbz, cbr, ar, tar
